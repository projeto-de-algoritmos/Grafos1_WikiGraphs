## Uso
1. Clone o projeto:
```
git clone https://github.com/projeto-de-algoritmos/Grafos1_WikiGraphs.git
```

2. Instale as dependências do projeto:
```
yarn
```
ou
```
npm install
```

3. Rode o projeto:
```
yarn start
```
ou
```
npm start
```
